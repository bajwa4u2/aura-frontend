# Aura Platform LLC Business Deck

This folder contains source material for the Aura Platform LLC business deck.

Current deck exports were removed because generated PDFs had layout and rendering issues. Future deck generation must be PDF-first, with fixed slide dimensions, bounded text areas, deterministic pagination, and export validation before any PDF is treated as final. Do not use responsive-web export as the production path.

## Canonical Positioning

- Aura = accountable public discourse and institutional communication infrastructure.
- Orchestrate = AI-assisted revenue automation and operational execution platform from outreach -> follow-ups -> meetings -> workflow execution -> billing.
- Aura Platform LLC = infrastructure company combining trusted communication and operational execution.

## Structure

- `source/aura_platform_business_deck_master.md` contains the controlled business deck source.
- `archive/` is reserved for intentionally retained references only. Broken exports, temporary screenshots, scripts, and duplicate generated files should not be kept.
